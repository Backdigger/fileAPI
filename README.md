```
To set up the project, do:

1. npm install
2. npm install -g gulp
3. npm install --save-dev gulp
4. gulp webserver

Technologies used:

- HTML 5 (File API)
- CSS 3
- JavaScript
- npm
- gulp


Task: Using File API 
- get images from local folders
- put them into <canvas> via File Reader
- get their rgb value in pixels
- filter the image via converting rgb to HSL color scheme, set to black and white
```
